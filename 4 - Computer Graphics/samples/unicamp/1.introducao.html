Introdução

Este tutorial apresenta alguns conceitos de computação gráfica utilizando uma das APIs de progração gráfica mais populares, o OpenGL. Foi concebido como material acessório para uma disciplina de computação gráfica da Universidade Estadual de Campinas (UNICAMP, CESET)) e do Centro Universitário Salesiano de São Paulo - UNISAL (Campinas,SP) e, assim sendo, assume que o aluno possui os fundamentos teóricos necessários para acompanhar cada lição. Entretanto, grande parte destes fundamentos é intuitivo para a maioria das pessoas, encorajando àqueles interessados no assunto a acompanharem as práticas e realizarem os exercícios.

A primeira versão deste Tutorial foi desenvolvida pelo Prof. Agostinho de Medeiros (UFRN), em 2000, e vem sendo modificada ao longo dos anos para melhor se adequar à disciplina de computação gráfica nos diferentes cursos de computação no que diz respeito à graduação.

Toda e qualquer sugestão e/ou contribuição visando melhorar e evoluir este tutorial será bemvinda. Pode mandá-la diretamente para mim via e-mail: magic@ceset.unicamp.br ou mcarvalho@unisal.com.br
