3.2. Outros programas relacionados

  1. polys.c. This program demonstrates polygon stippling.

    Programa polys.c

  2. smooth.c. This program demonstrates smooth shading. A smooth shaded polygon is drawn in a 2-D projection.

    Programa smooth.c

  3. picksquare.c. Use of multiple names and picking are demonstrated. A 3x3 grid of squares is drawn. When the left mouse button is pressed, all squares under the cursor position have their color changed.

    Programa picksquare.c
